# Grubtale

An Undertale-inspired GRUB theme that fills your boot process with **DETERMINATION**.

# Installation

## Compile the engine.
```
git clone "https://github.com/lazypwny751/grubtale.git" cd "grubtale"
make all
```

## Run self-install.
```
./grubtale -S -p "/boot/grub/themes" -s <systemd/sysvinit>
```

## Requirements

- gcc/clang
- make
- libcairo2
- stdlib
- getopt
